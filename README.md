🎯 Особенности
📮 Красивый конверт с бьющимся сердечком

💝 Поздравление с 8 марта при нажатии

🦎 36 уникальных аксолотлей в стиле Minecraft (ровно 6×6 сетка)

🎨 5 цветовых вариаций аксолотлей (розовые, коричневые, золотые, голубые, синие)

📱 Адаптивный дизайн для телефонов и планшетов

🎮 Пиксельная графика в стиле Minecraft

🚀 Как запустить локально
Скачайте файл index.html

Откройте его в любом браузере (Chrome, Firefox, Safari и т.д.)

Нажмите на конвертик и наслаждайтесь! 🎉

📦 Деплой на GitHub Pages
Создайте репозиторий на GitHub

Загрузите файл index.html

Включите GitHub Pages:

Settings → Pages

Source: Deploy from a branch

Branch: main → /root

Save
